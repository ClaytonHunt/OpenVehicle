# Safety and Regulatory Reference

This folder contains guidance notes and references related to safety engineering and legal compliance.

## Important Distinction

- **Legal/registration compliance** is jurisdiction-specific and administrative/inspection-based.
- **Crashworthiness** is an engineering and validation outcome.

A vehicle can pass a local registration pathway and still have poor crash performance.

## Current Position

The Open Vehicle Platform repository currently provides conceptual and fabrication-oriented guidance. It does **not** provide certified crash-performance claims.

Builders should pursue professional engineering review where possible.
